Square Market Multiple Item Embedding

Any link to Square Market with class sq-item-embed will be replaced with an embedded item. You only need to include the script tag once (but it's harmless if you include it more than once).

The embedded item will adapt to the size of its container, from a minimum width of 250px up to a maximum width of 560px:
